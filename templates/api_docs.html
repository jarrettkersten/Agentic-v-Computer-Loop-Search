Cora Code Investigator API

External API reference for integrating with the Cora Code Investigator. Use these endpoints to query the Cora PPM codebase programmatically from tools like Jira, n8n, or custom integrations.

Bearer Token — Required for query endpoints Admin Login — Required for dashboard endpoints Public — Health & branch listing
Query
POST /api/v1/query Unified query endpoint (sync/async) Bearer

The primary endpoint for external integrations (Jira, n8n, custom tools). Accepts a natural language query and returns the agentic search result. Supports both synchronous and asynchronous modes. Requires a Bearer token matching the EXTERNAL_API_KEY environment variable.

Headers

NameValueRequired
AuthorizationBearer <EXTERNAL_API_KEY>Required
Content-Typeapplication/jsonRequired

Request Body

NameTypeRequiredDescription
querystringRequiredNatural language question
branchstringOptionalADO branch to search. Use /api/branches to list available branches
asyncbooleanOptionalIf true, returns a job_id immediately (202). Default: false

Response (sync)

200 OK
{
  "success": true,
  "answer": "## Overview\n...",
  "metadata": {
    "branch": "supported_release-1.25.2",
    "files_read": 18,
    "iterations": 8,
    "searches": 14,
    "confidence": "high",
    "elapsed_seconds": 47.2,
    "input_tokens": 52000,
    "output_tokens": 9200
  }
}

Response (async)

202 Accepted
{
  "job_id": "a1b2c3d4-...",
  "poll_url": "/api/v1/query/a1b2c3d4-..."
}

Try it out

GET /api/v1/query/{job_id} Poll async query result Bearer

Polls for the result of an async query started via /api/v1/query. Returns the result when complete, or a processing status with progress info.

Path Parameters

NameTypeRequiredDescription
job_idstringRequiredJob ID returned by the async query

Response (completed)

200 OK
{
  "success": true,
  "answer": "## Overview\n...",
  "metadata": {...}
}

Response (still processing)

200 OK
{
  "status": "processing",
  "message": "Query is still being processed..."
}

Try it out

GET /api/branches List available ADO branches Public

Returns a list of branches from the Azure DevOps repository that have been indexed for code search. Uses the Code Search API facets with a Git Refs API fallback.

Response

200 OK
{
  "success": true,
  "branches": [
    "supported_release-1.25.2",
    "supported_release-1.25.1",
    "develop"
  ]
}
